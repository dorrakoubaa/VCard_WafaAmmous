# Interior Designer vCard

Welcome to the Interior Designer vCard project! This project is designed to showcase the work and services of an interior designer through a modern and responsive web application.

## Project Structure

The project is organized as follows:

```
interior-designer-vcard
├── src
│   ├── index.html          # Main HTML document
│   ├── main.tsx           # Main TypeScript file for React
│   ├── components          # Contains all React components
│   │   ├── Header.tsx     # Displays the designer's name and title
│   │   ├── About.tsx      # Presents the designer's biography
│   │   ├── Services.tsx   # Outlines the services offered
│   │   ├── Portfolio.tsx   # Showcases past projects
│   │   └── Contact.tsx    # Provides contact information and form
│   ├── styles              # Contains CSS styles
│   │   └── main.css       # Main stylesheet for the vCard
│   ├── data                # Contains structured data
│   │   └── bio.json       # JSON file with biography and services
│   └── types               # TypeScript interfaces and types
│       └── index.ts       # Type definitions for the application
├── package.json            # npm configuration file
├── tsconfig.json           # TypeScript configuration file
└── README.md               # Project documentation
```

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd interior-designer-vcard
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the application**:
   ```
   npm start
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the vCard.

## Features

- **Responsive Design**: The vCard is designed to be mobile-friendly and looks great on all devices.
- **Showcase Projects**: Easily display your past work and projects in the portfolio section.
- **Contact Form**: Potential clients can reach out directly through the contact form.
- **Customizable**: Modify the components and styles to fit your personal branding.
